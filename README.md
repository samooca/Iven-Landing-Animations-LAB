# README

## Objetivo

Este repositório contém a estrutura, documentação e arquivos necessários para desenvolver Landing Pages utilizando IA, automações e ferramentas de desenvolvimento.

O projeto concentra:

* código;
* documentação;
* configurações;
* prompts;
* agentes;
* workflows;
* decisões técnicas.

---

## Finalidade

O repositório deve funcionar como uma base central de conhecimento do projeto.

Todas as informações importantes devem estar documentadas para facilitar:

* continuidade do desenvolvimento;
* manutenção futura;
* uso por agentes de IA;
* recuperação de contexto.
