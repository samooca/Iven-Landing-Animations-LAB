# Iven-Landing-Animations-LAB
Ambiente focado em landing pages premium com animações avançadas, unindo IA local via Ollama ao VS Code. O sistema otimiza todo o ciclo de desenvolvimento: desde a engenharia reversa de sites de referência até o design de interação, motion, programação, otimização, documentação e deploy, impulsionando a arquitetura completa do projeto.
